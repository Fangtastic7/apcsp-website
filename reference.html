Name Tag Description Example
List ls Shows you the files in the directory. ls temp

Nano nano Serves as a text editor for typing in files. nano readme
Remove rm Removing a file. rm readme readme2
Move mv Moving a file. mv readme tmp
Copy Files cp Copying a file cp tmp tmp3
Clear clear Clears the screen of code. clear
Less less Lets the user go up the file when space is pressed. less services
More more Lets the user go down in the file when space is pressed. more services
